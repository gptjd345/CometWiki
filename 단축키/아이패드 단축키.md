
1. 커서 이동 : cmd 화살표 (🔼 🔽 ◀️ ▶️)
	* 영역 선택  : shift cmd 화살표 (🔼 🔽 ◀️ ▶️)

2. 화면 끄기 : cmd ctrl q
3. 실행중인 앱목록 이동 : 🌍(지구본 or fn키) ⬆️
	* 🌍 + C : 제어센터(우측상단 내리면 뜨는 창 말함)
	* 🌍 + N :  잠금화면 내렸다가 올리기(잠깐만 ! 지금 몇시지?)
	* 🌍 + Q : 빠른 메모
	* 🌍 + D : 독 바 보이기/ 없애기
	* 🌍 + H : 홈화면으로 이동 
	* 🌍 + ◀️ / ▶️ : 이전 / 이후 앱 이동 (ex :홈화면으로 갔다가 빠르게 다시 돌아오기)
